# <img width="64" height="64" alt="Tyrian_Lens" src="https://github.com/user-attachments/assets/f9ca39f0-b477-40d3-8643-11c2534a6346" /> Tyrian Lens

A lightweight, borderless Guild Wars 2 companion app and interactive map designed for a seamless secondary-monitor experience. 

Tyrian Lens bridges the gap between the game and your browser, offering real-time character tracking and full account transparency without the clutter of a standard web browser.

##  Key Features

* **Real-Time Map Tracking:** Watch your character move across Tyria in real-time using Mumble Link integration.
* **Borderless Experience:** Custom-built window management for a sleek, modern look that sits perfectly on your second monitor.
* **Account Management:** Direct access to your inventory, bank, characters, and trading post data via the Guild Wars 2 API.
* **Privacy Focused:** Your API keys are stored locally on your machine in `%LocalAppData%\TyrianLens`. We never see your data.

##  Installation

1.  Head over to the [Releases](https://github.com/Yoast79/Tyrian-Lens/releases) page.
2.  Download the latest **Tyrian_Lens_Setup.exe**.
3.  Run the installer. If Windows flags it with a "SmartScreen" warning, click **More Info** -> **Run Anyway** (this occurs because the app is independently developed and not yet digitally signed).
4.  Launch **Tyrian Lens** from your desktop!

##  First-Time Setup

To unlock the full potential of the app's functionality:
1.  Open the app and click the **Key Icon** - **API Management**.
3.  Add an API key from [account.arena.net/applications](https://account.arena.net/applications). 
    * *Recommended permissions: ALL THE THINGS!.*
